차 A→B 레이싱 프레임을 벗어나, "못 걷던 생명체가 스스로 걷게 되는" 진화 시뮬.
보는 사람이 "진짜 AI가 학습한다"고 느끼게 하는 3대 UI 탑재.

## 컨셉
- 생명체 = 4개 노드(점질량) + 6개 근육(K4 완전그래프 스프링).
- 각 근육 길이를 작은 신경망(뇌)이 실시간 제어 → 리듬(위상시계)으로 걸음새 창발.
- 적합도 = 앞으로 간 거리. 신경진화(GA: 엘리트+토너먼트+돌연변이+이민자).
- 물리: 반암시적 오일러 + 지면 마찰(디딘 발만 마찰 → 추진력). numpy 벡터화로 떼 동시 평가.

## 화면/기능
- [ ] 메인 아레나: 떼 전체 흐릿 + 선두 생명체 강조, 카메라가 선두 추적, 거리 눈금 스크롤
- [ ] 살아있는 신경망: 선두의 뇌 노드/엣지 실시간 점등(입력→은닉→출력)
- [ ] 실시간 학습 곡선: 세대별 최고/평균 적합도 그래프(자동 스케일)
- [ ] 1세대 vs 현재 분할화면: 1세대 챔피언(부들부들) vs 현재 챔피언(성큼) 동시 반복재생
- [ ] HUD: 세대/누적시도/최고거리, 조작 안내(SPACE 재학습, ESC 종료)

## 구현 단계
1. [ ] 생명체 상수(노드/근육/L0) + Walkers 클래스(상태·센서·뇌·물리·진화)
2. [ ] 학습 루프(프레임당 1스텝, T스텝/세대 후 진화, 히스토리 기록)
3. [ ] 챔피언 캡처(1세대·현재) + 분할 재생 스웜(2)
4. [ ] 렌더: 메인 아레나 → 신경망 → 학습곡선 → 분할화면 → HUD
5. [ ] 물리 튜닝(폭발 방지: 속도 클램프/감쇠, 마찰 계수) — 실제로 전진하는지 검증
